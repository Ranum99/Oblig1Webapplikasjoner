Oppgave 1
=========

List with 5 user stories:

1. As a student, I want to be able to see the result at the end of the exam, so that I know if i failed or not.
2. As a student, I want to be able to login to my/register an account, so that I can take my exams.
3. As a student, I want to be able to see how long time I have left, so that the exam just suddenly ends.
4. As an admin, I want to be able to save an exam as a template, so that I don't have to make an exam from scratch every time.
5. As an admin, I want to be able to send out an emil to students who have finished the course successfully, so that the students can take the exam.
